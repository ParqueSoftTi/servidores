Ve al sitio web oficial de Apache: httpd.apache.org y descarga la última versión estable de Apache para Windows.

Una vez que hayas descargado el archivo de instalación, ábrelo y sigue las instrucciones del instalador. Durante el proceso de instalación, se te pedirá que selecciones componentes y opciones de configuración. Asegúrate de seleccionar las opciones que se ajusten a tus necesidades.

Después de instalar Apache, es posible que desees configurar algunas opciones. El archivo de configuración principal de Apache es httpd.conf, que generalmente se encuentra en la carpeta conf dentro del directorio de instalación de Apache. Aquí puedes modificar opciones como los puertos en los que Apache escucha, los directorios de documentos, la configuración de seguridad, entre otras.

Cambiar el puerto:

Por defecto, Apache escucha en el puerto 80. Si este puerto está siendo utilizado por otro servicio en tu sistema, puedes cambiarlo a un puerto diferente. Por ejemplo, para cambiarlo al puerto 8080, puedes buscar la línea que dice:

Listen 80

Cambiar por

Listen 8080

Configurar los directorios de documentos

Puedes especificar la ubicación de los archivos que Apache debe servir. Por ejemplo, si deseas cambiar el directorio raíz de documentos de htdocs a C:/mi_sitio_web, puedes buscar la línea que dice:

DocumentRoot "ruta/a/htdocs"

y cambiarla a:

DocumentRoot "C:/mi_sitio_web"

También debes cambiar la directiva para que apunte al nuevo directorio:

<Directory "C:/mi_sitio_web">

Configuración de seguridad:

Puedes aplicar reglas de seguridad básicas, como restringir el acceso a ciertas carpetas o habilitar/deshabilitar módulos de Apache según sea necesario. Por ejemplo, para permitir el acceso solo desde la red local, puedes agregar lo siguiente:

<Directory "C:/mi_sitio_web">
    Require local
</Directory>

Una vez que hayas configurado Apache según tus preferencias, puedes iniciar el servicio. Puedes hacerlo desde el Panel de Control de Windows, buscando "Servicios", o ejecutando services.msc. Busca "Apache HTTP Server" en la lista de servicios, haz clic derecho y selecciona "Iniciar".

Para verificar que Apache se haya instalado y esté funcionando correctamente, abre un navegador web y escribe localhost en la barra de direcciones. Deberías ver la página de inicio predeterminada de Apache o alguna otra página que hayas configurado como la página de inicio.
